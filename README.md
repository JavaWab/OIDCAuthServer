# OIDCAuthServer
##简介
本项目是一个实现IODC协议的认证服务。本身属于Spring Cloud为服务，依赖zookeeper、redis、mongodb，并且拓展实现了MongodbTokenStore、DefaultOAuth2ProviderTokenService等，可直接用于实现第三方登录。
支持OIDC（以及Oauth2）协议的各种模式。
